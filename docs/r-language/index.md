# 📊 R语言学习路径

> 从Python过渡到R，掌握生物统计和数据可视化

---

## 学习指引

这个路径假设你已经完成了**Python基础学习**。你会发现：

- ✅ Python学的编程思维 → 直接迁移到R
- ❗ 语法不同 → 对照学习
- 🎯 重点在**统计**和**可视化** → R的强项

## 课程目录

| 章节 | 内容 | 前置要求 |
|------|------|---------|
| 第1章 | 从Python到R — 对比学习法 | Python基础 |
| 第2章 | R基础语法 — 数据类型、向量、数据框 | 第1章 |
| 第3章 | 数据可视化 — ggplot2入门 | 第2章 |
| 第4章 | 生物统计实战 — t检验、方差分析、相关性 | 第3章 |

## 学习方法建议

1. **对照学** — 看到R的语法，想想Python里对应的写法
2. **重点在可视化** — R的ggplot2是画出发表级图表的最强工具
3. **练统计** — 结合你的医学统计知识，用R实现
