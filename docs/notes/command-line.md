# 命令行基础

> 常用终端命令速查

---

## 常用命令

| 命令 | 作用 | 例子 |
|------|------|------|
| `pwd` | 查看当前路径 | `pwd` → `/c/Users/YizL` |
| `ls` | 列出文件 | `ls -la`（显示所有文件） |
| `cd` | 切换目录 | `cd /d/Desk/编程入门` |
| `mkdir` | 创建文件夹 | `mkdir test` |
| `cp` | 复制 | `cp a.txt b.txt` |
| `mv` | 移动/重命名 | `mv a.txt 新名.txt` |
| `rm` | 删除 | `rm test.txt` |
| `python` | 运行Python | `python hello.py` |

## pip命令

| 命令 | 作用 |
|------|------|
| `pip install 包名` | 安装Python包 |
| `pip list` | 查看已安装包 |
| `pip uninstall 包名` | 卸载包 |

!!! warning "小心 rm"
    `rm` 删除的文件不会进回收站，慎用！
