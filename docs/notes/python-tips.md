## 常见错误

### 1. 索引从0开始忘记

```python
lst = [10, 20, 30]
print(lst[1])  # 期望第一个，实际得到20！
```

**解决**：记住 `lst[0]` 才是第一个元素。

### 2. 字符串拼接时忘记转类型

```python
age = 25
print("年龄：" + age)  # ❌ TypeError
print("年龄：" + str(age))  # ✅ 正确
```

### 3. 缩进错误

```python
if True:
print("缩进不对")  # ❌ IndentationError
```

**解决**：Python用缩进表示代码块，保持一致（通常4个空格）。
