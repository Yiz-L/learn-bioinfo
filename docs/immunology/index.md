# 🔬 编程与免疫学

> 让编程成为免疫学研究的得力助手

---

## 为什么编程对免疫学很重要

现代免疫学研究越来越依赖计算分析：

| 研究方向 | 涉及的数据分析 | 常用工具 |
|---------|--------------|---------|
| 单细胞测序 | 10万+细胞的基因表达矩阵 | Python (Scanpy) |
| 流式细胞术 | 多通道荧光数据 | R (flowCore) |
| 免疫组库分析 | TCR/BCR序列 | Python + R |
| 细胞因子检测 | ELISA/MSD数据处理 | Python/Excel替代 |
| 文献计量分析 | PubMed文献挖掘 | Python (Biopython) |

---

## 学习路线

1. **Python基础** → 能处理数据、写脚本
2. **R基础** → 能统计、可视化
3. **专用工具** → 根据研究方向选学

此部分内容将随着你的编程水平提高逐步填充。
