# 🐍 Python学习路径

> 从零开始，学会编程思维，写出第一个科研脚本

---

## 学习指引

本路径假设你**没有任何编程基础**，每一步都从最基础讲起。

## 课程目录

| 章节 | 内容 | 状态 |
|------|------|------|
| 第1章 | 你好，编程世界 — 安装、第一个程序、打印输出 | ⏳ 待开始 |
| 第2章 | 变量与数据 — 数字、字符串、输入输出 | ⏳ 待开始 |
| 第3章 | 条件与循环 — if、for、while | ⏳ 待开始 |
| 第4章 | 数据结构 — 列表、字典、元组、集合 | ⏳ 待开始 |
| 第5章 | 函数与模块 — 函数、导入、pip | ⏳ 待开始 |
| 第6章 | 文件与数据 — 读写文件、CSV处理 | ⏳ 待开始 |
| 第7章 | 实战——科研小工具 | ⏳ 待开始 |

## 学习方法建议

1. **读** - 阅读每章的概念讲解
2. **敲** - 亲手在VSCode里打出每一行代码
3. **改** - 修改代码里的数字/文字，看看会怎样
4. **用** - 想想今天学的能用在你的什么场景

!!! tip "不要只看不练"
    编程就像学骑自行车，看再多教程都不如亲手骑一段路。**每行代码都要自己敲一遍！**
