## 1.1 程序是什么？

想象你给一个非常听话（但很笨）的外国人写做菜步骤：

```
1. 打开冰箱
2. 拿出鸡蛋
3. 热锅
4. 倒油
5. 煎蛋
```

**程序就是**写给电脑的"做菜步骤"。电脑比外国人还听话，但也比外国人还笨——它只会严格按照你写的步骤执行。

---

## 1.2 你的第一行代码

打开 VSCode，新建一个文件，命名为 `hello.py`，输入：

```python
print("你好，免疫学！")
```

然后在终端里运行：

```bash
python hello.py
```

你会看到：

```
你好，免疫学！
```

!!! success "恭喜！你写下了人生中第一行代码！"

### 代码拆解

| 部分 | 含义 |
|------|------|
| `print` | 告诉电脑："我要打印东西" |
| `(...)` | 括号括住要打印的内容 |
| `"你好，免疫学！"` | 要打印的文字（字符串） |

## 1.4 常见错误（别怕！）

```python
print("少了一个引号)
```

你会看到：

```
SyntaxError: unterminated string literal
```

**不要慌！** 错误信息是编程的常态。它只是告诉你"少了一个引号"。加上就好了。

```python
print("现在好了")
```

!!! tip "错误是你的朋友"
    每个程序员每天都会看到无数错误信息。学会读错误信息，比学会写代码还重要！

## 1.5 医学小剧场

想象你在整理病例数据：

```python
print("患者编号：001")
print("诊断：类风湿关节炎")
print("治疗方案：TNF-α抑制剂")
```

用几行 `print` 就可以快速生成结构化的数据条目。
