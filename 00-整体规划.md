# 医学僧编程学习路径网站 — 总体规划

## Context

**用户背景**：医学生，免疫学方向，未来研究生阶段需要编程辅助科研。完全零基础编程和Web开发。

**核心需求**：搭建一个**个人使用的编程学习路径网站**，以Python入门，过渡到R语言，逐步扩展到生物信息学，同时集成科研任务管理和笔记系统。

**为什么现在做**：先规划好学习路径和工具平台，让编程学习和专业学习同步进行，避免"学了用不上"或"要用时不会"的脱节。

---

## 一、技术栈推荐：MkDocs + Material Theme

### 选型理由

| 维度 | MkDocs + Material | 其他方案（Flask/Next.js等） |
|------|-------------------|---------------------------|
| **学习成本** | ⭐ 零Web基础也能上手 | 需要先学HTML/CSS/JS/Python框架 |
| **内容编写** | ⭐ Markdown写作，专注内容 | 需要写模板、组件 |
| **Python生态** | ⭐ `pip install` 即可 | Flask是Python但需要Web知识 |
| **开箱即用** | ⭐ 搜索、导航、主题、代码高亮 | 全部要自己搭建 |
| **可扩展性** | ⭐ 插件系统（Python编写） | 灵活但复杂 |
| **部署** | ⭐ GitHub Pages 免费一键部署 | 需要服务器 |

### 核心优势：与你的学习路径完美对应

```
你学Python → 你就学会了如何配置/扩展这个网站！
├── pip install → 安装MkDocs和插件
├── Python语法 → 写自定义插件、自动化脚本
├── Markdown → 写教程内容、笔记
├── Git/GitHub → 版本管理、部署
└── YAML配置 → 网站设置、导航结构
```

> **每个Python知识点都能在这个网站上找到应用场景**，学以致用。

## 三、实施阶段

### 阶段一：骨架搭建（动手第一天就能上线）

**目标**：把网站跑起来，看到效果

1. 安装Python（如果还没装）
2. `pip install mkdocs-material`
3. 初始化项目骨架
4. 配置导航结构（上面的框架）
5. 写首页：欢迎页 + 学习路线图
6. 部署到 GitHub Pages（免费）

**输出物**：一个能看到导航、能搜索、有内容框架的在线网站

### 阶段二：Python学习路径内容（核心）

**目标**：逐章填充Python教程

每章结构：
```
┌─ 概念讲解（为什么学这个）─────┐
│  用医学生能懂的方式解释          │
├─ 代码示例 ──────────────────────┤
│  带注释的代码，可直接复制运行      │
├─ 练习（你来做） ────────────────┤
│  给提示不给答案，但本站有答案页    │
├─ 免疫学小应用 ──────────────────┤
│  "这个知识点能用在XX研究场景"     │
└─ 检查清单 ──────────────────────┘
  ✓ 我理解了概念
  ✓ 我亲手敲了代码
  ✓ 我完成了练习
```

### 阶段三：个人工具集成（边学边造）

**目标**：用刚学的Python知识给网站加功能

1. **任务管理器** — 简单的Python脚本（CLI版），读写任务列表
2. **笔记速查** — 利用MkDocs搜索功能整理笔记
3. **学习计时器** — Python脚本记录每天学习时长
4. **进度统计** — 统计学了多少章、多少练习

### 阶段四：R语言路径（Python基础扎实后）

**目标**：用Python建立的编程思维快速上手R

- 用对比学习法（Python的for循环 ↔ R的for循环）
- 侧重统计和可视化（ggplot2）
- 免疫学数据分析案例

### 阶段五：持续进化（长期）

- 加入更多免疫学生信案例
- 整合实际研究数据
- 根据研究生阶段需求随时扩展

## 四、关键技术决策

| 决策项 | 选择 | 原因 |
|--------|------|------|
| 静态站点生成 | MkDocs | Python生态，零Web基础可上手 |
| 主题 | Material for MkDocs | 最流行的MkDocs主题，功能齐全 |
| 部署方式 | GitHub Pages | 免费，零运维 |
| 内容格式 | Markdown | 最简写作语法 |
| 版本管理 | Git | 本身就是重要技能 |
| 代码运行 | 本地（教程引导安装） | 学会真实环境配置 |
| 域名 | `你的用户名.github.io/learn-bioinfo` | 免费 |

## 五、为什么这个方案适合你（零基础友好）

```
你现在的状态
│
├── ❌ 不会写HTML/CSS/JS → MkDocs不需要你写
├── ❌ 不懂服务器运维 → GitHub Pages全托管
├── ❌ 没有Web开发概念 → Markdown + 配置 = 网站
├── ✅ 会用浏览器 → 开箱即看的网站
└── ✅ 即将学Python → 学会了就能改造自己的网站
         │
         ▼
    上手即用，随着你的编程能力成长，网站也越来越强大
```
